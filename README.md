# 🩸 LifeLink - Blood Bank & Donation Platform

LifeLink is a modern, premium, and fully responsive web platform designed to streamline blood donation networks and bridge the gap between blood donors, emergency seekers, hospitals, and donation camps in real-time. Built with **React**, **Vite**, and **Tailwind CSS**, it features a clean, medical-minimalist design language.

---

## 🌟 Key Features

*   **🔍 Find Donors**: Connects users in need of blood with active, registered donors, searchable by blood group and location.
*   **🚨 Request Blood**: Post emergency blood requirements specifying blood type, hospital, units needed, and urgency level.
*   **📅 Donation Camps**: Find, explore, and register for active blood donation camps in your vicinity.
*   **🏥 Affiliated Hospitals**: Browse partner hospitals and clinics integrated into the LifeLink vitality system.
*   **✍️ Easy Registration**: Sign up as a donor, volunteer, or hospital partner to help save lives.
*   **📝 Informative Blog**: Access educational articles, tips, and insights on the benefits and guidelines of blood donation.

---

## 🛠️ Technology Stack

*   **Frontend Library**: React 18
*   **Build Tool**: Vite (Fast HMR & Optimized Bundling)
*   **Styling**: Tailwind CSS (with customized theme variables for Material Design)
*   **Routing**: React Router DOM
*   **Icons & Fonts**: Google Fonts (Public Sans) & Material Symbols Outlined

## 🚀 Getting Started

Follow these steps to run the project locally on your machine:

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed (version 16.x or higher recommended).

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/codesrishti/LifeLink-Blood-Bank-Platform.git
   cd LifeLink-Blood-Bank-Platform
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

### Running Locally

To start the development server:
```bash
npm run dev
```
Open your browser and navigate to `http://localhost:5173` (or the port specified in your terminal).

### Production Build

To build the project for production:
```bash
npm run build
```
The optimized production bundle will be generated in the `dist` directory.

## 🎨 Design System & Styling

LifeLink uses a custom **Material Minimalist** design system configured inside Tailwind CSS (`src/index.css`):
*   **Primary Accent**: Blood Red (`#af101a`)
*   **Secondary Accent**: Soft Slate (`#6b5a60`)
*   **Tertiary Accent**: Medical Blue (`#0058a2`)
*   **Background**: Clean Light Ice (`#f7f9ff`)
*   **Shadows**: Soft medical shadows (`medical-soft-shadow`) and smooth translation hover animations (`card-hover-lift`) for interactive components.
